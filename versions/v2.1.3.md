# v2.1.3

Дата: 2026-04-11
Статус: стабильная версия

## Что изменено

- Реализован `Crash Logging and Diagnostics`: добавлен `src/limuzin_grid_manager/app/runtime.py`, который выбирает каталог логов в локальном app-data, настраивает `RotatingFileHandler` и хранит runtime-журнал в `runtime.log`.
- `src/limuzin_grid_manager/__main__.py` теперь включает runtime-логирование и глобальные hooks для необработанных исключений до показа `MainWindow`.
- Worker-ы в `src/limuzin_grid_manager/ui/main_window.py` и `src/limuzin_grid_manager/ui/points_window.py` больше не теряют traceback: при падении фонового экспорта или импорта полная трассировка уходит в лог, а пользователю показывается путь к журналу.
- Добавлены `tests/test_runtime.py` и новый UI-регрессионный сценарий в `tests/test_ui.py`, который проверяет логирование traceback и user-facing diagnostics при падении Excel import worker-а.
- Синхронизированы version metadata и документы релизной линии `v2.1.3`.
