# v2.0.0 - Стабилизация и релизная подготовка

Дата: 2026-04-08

Статус: стабильная версия, принята пользователем после ручного теста и подготовлена к публикации.

## Что изменено

- Версия обновлена до `2.0.0` в `pyproject.toml`, `src/limuzin_grid_manager/__init__.py` и `version_info.txt`.
- `v2.0.0` фиксирует как единый major-релиз итог этапов `v1.1.0`-`v1.6.0`: KML/ZIP, SVG, GeoJSON, CSV, темы интерфейса, многозонный экспорт и SVG-полировку подписей `100x100`.
- Обновлены `README.md`, `USER_GUIDE.md`, `GRIDBASE.md`, `GITHUB.md`, `ROADMAPv2.md` и индекс версий.
- Новая функциональность сверх уже подготовленных этапов `v1.1.0`-`v1.6.0` в `v2.0.0` не добавляется: этап предназначен для стабилизации и релизной проверки.

## Что проверено

- `uv lock --offline`: lock-файл обновлен с `1.6.0` на `2.0.0`.
- `uv run --offline --extra dev pytest`: `69 passed`.
- `uv run --offline --extra dev python -m compileall src tests`: компиляция прошла без ошибок.
- `$env:UV_OFFLINE='1'; .\build_exe_windows.bat`: EXE собран успешно.
- `dist/LIMUZIN_GRID_MANAGER.exe` существует, `FileVersion` и `ProductVersion` равны `2.0.0.0`.
- Smoke-запуск EXE пройден: процесс стартует и не завершается в первые 3 секунды.
- При сборке сохранилось прежнее предупреждение PyInstaller hook-pyproj: `Datas for pyproj not found at: ...\.venv\Library\share\proj`.
- Временные build/cache-артефакты после проверки удалены; `dist/` оставлен для ручного теста и игнорируется git.
- Пользователь вручную протестировал текущую версию и подтвердил, что внесенные изменения работают.
