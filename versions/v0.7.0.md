# v0.7.0 — Сохранение проекта и пресеты

Дата: 2026-04-07

Статус: рабочая версия.

## Что добавлено

- Добавлен файл проекта `.lgm.json`.
- Проект сохраняет:
  - координаты `NW` и `SE`;
  - включенные сетки `1000x1000` и `100x100`;
  - схему нумерации `100x100`;
  - режим округления;
  - пользовательские имена больших квадратов;
  - KML-стиль, палитры и заливки;
  - формат экспорта, папку и имя файла.
- Добавлено открытие сохраненного проекта без потери настроек.
- Добавлены встроенные пресеты:
  - режим `v0.0.1`;
  - схемы нумерации `100x100`;
  - KML-стили.

## Архитектура

- Добавлен модуль `src/limuzin_grid_manager/app/project.py`.
- JSON-сериализация проекта вынесена из `MainWindow`.
- Файл проекта имеет схему `limuzin-grid-manager-project` и версию схемы `1`.
- Для неподдерживаемых или поврежденных файлов показывается понятная ошибка вместо молчаливой загрузки.

## Интерфейс

- Добавлено меню `Проект` с действиями:
  - `Новый проект`;
  - `Открыть`;
  - `Сохранить`;
  - `Сохранить как`.
- Добавлено меню `Пресеты`.
- Добавлена верхняя панель быстрых действий, чтобы сохранение и пресеты были видны без поиска в меню.
- Пресеты применяются к текущему проекту и не меняют координаты.
- В верхней панели показывается текущий файл проекта.
- Вкладка `Экспорт` теперь прокручивается при небольшом размере окна и не сжимает блоки до нечитаемого состояния.

## Что проверено

- Unit-тесты ядра и экспорта проходят.
- Добавлены тесты сохранения и открытия проекта.
- Добавлены тесты нормализации имени `.lgm.json` и ошибки для неподдерживаемой схемы.
- Добавлена smoke-проверка панели проекта и применения пресета в `MainWindow`.
- Проверена компиляция `src` и `tests`.
- Собран `dist/LIMUZIN_GRID_MANAGER.exe`.
- Версия EXE: `0.7.0.0`.
- Ручной тест версии пользователем пройден.
