## What Changed

- Implemented `Responsive Excel Import` in `src/limuzin_grid_manager/ui/points_window.py`.
- Added `PointsImportWorker` on a dedicated `QThread`, so loading `.xlsx` now happens outside the GUI thread.
- Added explicit import-running UX in the `Точки из Excel` window: the progress bar is shown in indeterminate mode, conflicting controls are disabled, and window closing is blocked until import finishes.
- Kept workbook parsing and validation rules unchanged by leaving `src/limuzin_grid_manager/app/point_import.py` untouched.
- Extended `tests/test_ui.py` with smoke coverage for asynchronous import, running-state disabling, and close blocking while import is active.
- Synced version metadata and release documents to `2.1.2`.
