# GRIDVERSIONS: LIMUZIN GRID MANAGER

Этот файл хранит краткую историю версий LIMUZIN GRID MANAGER.

Для каждой новой версии нужно:

1. Добавить строку в таблицу ниже.
2. Создать отдельный файл в этой же папке: `vX.Y.Z.md`.
3. В отдельном файле описать изменения подробнее: что добавлено, что исправлено, что проверено.
4. Обновить версию в `pyproject.toml`, `src/limuzin_grid_manager/__init__.py` и `version_info.txt`.
5. Запустить тесты и, если нужен новый EXE, пересобрать `dist/LIMUZIN_GRID_MANAGER.exe`.

## Индекс версий

| `v0.0.1` | 2026-04-07 | Рабочая версия | Первый PySide6-релиз: сетки 1000x1000/100x100, СК-42 -> WGS84, KML/ZIP, без цветной заливки. |

## Правила нумерации

Используем формат:

```text
vMAJOR.MINOR.PATCH
```

- `MAJOR` — крупные несовместимые изменения.
- `MINOR` — новые функции без поломки текущего поведения.
- `PATCH` — исправления, мелкие улучшения, документация.

Для текущего проекта стартовая рабочая версия: `v0.0.1`.
