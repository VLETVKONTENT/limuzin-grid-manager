# v0.8.0 — Устойчивость, производительность и крупные области

Дата: 2026-04-07

Статус: рабочая версия.

## Что добавлено

- Добавлены предупреждения для больших экспортов: приложение заранее показывает, что будет записано много KML-объектов.
- Добавлен лимит для чрезмерно больших экспортов, чтобы не создавать файл, который с высокой вероятностью сломает интерфейс или будет непрактичен в AlpineQuest.
- Добавлена оценка количества KML-объектов и примерного размера результата во вкладке `Экспорт` и на вкладке `Проверка`.
- Добавлена проверка свободного места перед записью результата.
- Добавлена отмена долгого экспорта из нижней панели и вкладки `Экспорт`.
- Экспорт теперь пишет результат во временный файл рядом с целевым и заменяет итоговый файл только после успешного завершения.

## Производительность и прогресс

- Прогресс экспорта теперь считается по реальным KML-объектам: большой квадрат и каждый малый `100x100` учитываются отдельно.
- Сигналы прогресса ограничены по частоте на очень больших экспортах, чтобы не перегружать UI очередью обновлений.
- Для больших областей предпросмотр получает отдельное предупреждение: плотные линии и подписи могут скрываться, чтобы окно оставалось отзывчивым.
- Диалог имен больших квадратов защищен от открытия таблицы на слишком большом количестве `1000x1000`, чтобы интерфейс не зависал.

## Ошибки и устойчивость

- Ошибки ввода координат стали точнее: сообщение указывает конкретное поле `NW X`, `NW Y`, `SE X` или `SE Y`.
- При отмене экспорта временный незавершенный файл удаляется, а старый результат не заменяется.
- Если места на диске недостаточно, пользователь получает понятное сообщение с примерным требуемым и доступным объемом.
