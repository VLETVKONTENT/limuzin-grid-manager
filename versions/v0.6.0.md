# v0.6.0 — Отдельная вкладка экспорта

Дата: 2026-04-07

Статус: рабочая версия.

## Что добавлено

- Добавлена отдельная вкладка `Экспорт` в правой рабочей области.
- Вкладка показывает:
  - выбор текущего варианта экспорта;
  - выбор папки результата;
  - выбор имени файла;
  - системный диалог выбора полного пути файла;
  - сводку перед запуском экспорта.
- Для KML показывается, что будет создан один общий `.kml`.
- Для ZIP показывается, что будет создан один `.zip` и сколько `tile_###.kml` будет внутри архива.
- Добавлен блок для будущих вариантов экспорта: только большие квадраты, только малые квадраты, таблица номеров, настройки проекта и пресеты.

## Архитектура

- Добавлен модуль `src/limuzin_grid_manager/app/export_formats.py`.
- Описание доступных экспортных вариантов, расширений файлов, стандартных имен, фильтров диалогов и пользовательской сводки вынесено из `MainWindow`.
- Текущие сценарии `KML` и `ZIP` продолжают идти через существующий `ExportMode`, `export_grid()`, потоковый KML и ZIP по большим квадратам.

## Интерфейс

- Блок экспорта убран из левой панели, чтобы основной экран оставался для координат, сетки, имен квадратов и KML-стиля.
- Кнопка генерации осталась в нижней панели и продублирована во вкладке `Экспорт`.
- При смене KML/ZIP имя файла автоматически получает правильное расширение.
- Если файл результата уже существует, перед перезаписью показывается подтверждение.
