# v2.2.0

Дата: 2026-04-12
Статус: стабильная версия

## Что изменено

- Реализован этап `Signed Public Release Gate`: `build_exe_windows.bat` по умолчанию по-прежнему собирает unsigned EXE для локальной разработки, но теперь умеет подписывать `dist/LIMUZIN_GRID_MANAGER.exe` через Authenticode при наличии `LGM_SIGN_EXE=1`, `LGM_SIGN_PFX_PATH` и `LGM_SIGN_PFX_PASSWORD`.
- Скрипт сборки делает fast-fail preflight для signing-конфига: до запуска `uv` и PyInstaller проверяет наличие PFX-файла, пароля и `signtool.exe`, а после подписи сразу валидирует результат через `Get-AuthenticodeSignature`.
- `.github/workflows/release-windows.yml` сохраняет воспроизводимую unsigned-сборку `LIMUZIN_GRID_MANAGER.exe` в GitHub Actions как artifact, а optional signing проверяется локально через `build_exe_windows.bat`.
- Обновлены `GITHUB.md`, `README.md`, `USER_GUIDE.md`, `GRIDBASE.md`, `roadmap.md`, `versions/GRIDVERSIONS.md` и `FIXPROD.md`, чтобы release-процесс различал локальную smoke-проверку, зеленый CI, наличие артефакта и отдельную обязательную проверку статуса подписи `Valid`.
- Синхронизированы version metadata для рабочей линии `v2.2.0`, включая `pyproject.toml`, `src/limuzin_grid_manager/__init__.py`, `version_info.txt` и `uv.lock`.
- После ручного пользовательского теста версия принята как стабильная; проект сохраняет optional signing-flow, но больше не требует покупки публичного коммерческого code-signing сертификата.

## Примечание
