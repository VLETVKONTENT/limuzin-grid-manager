# v2.1.4

Дата: 2026-04-11
Статус: стабильная версия

## Что изменено

- Реализован этап `CI and Release Discipline`: добавлены `.github/workflows/ci.yml` и `.github/workflows/release-windows.yml`.
- `ci.yml` теперь выполняет Windows-проверку для push и pull request: устанавливает Python и `uv`, синхронизирует dev-зависимости, запускает `pytest`, `python -m compileall src tests` и `uv build`.
- `release-windows.yml` запускается вручную или по тегу `v*`, пересобирает `dist/LIMUZIN_GRID_MANAGER.exe` через `build_exe_windows.bat` и публикует EXE как GitHub Actions artifact без этапа подписи.
- Обновлены `GITHUB.md`, `README.md`, `GRIDBASE.md`, `roadmap.md` и индекс версий, чтобы release-процесс теперь явно различал локальную smoke-проверку, зеленый CI и наличие собранного release-артефакта.
- Синхронизированы version metadata для рабочей линии `v2.1.4`, включая `pyproject.toml`, `src/limuzin_grid_manager/__init__.py`, `version_info.txt` и `uv.lock`.
