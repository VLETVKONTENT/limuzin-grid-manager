# v0.2.0: спиральная нумерация 100x100

Дата: 2026-04-07

Статус: принята после ручной проверки.

## Что добавлено

- Добавлены новые схемы нумерации малых квадратов `100x100`:
  - `спираль от центра наружу`;
  - `спираль от края к центру`.
- Добавлена настройка направления спирали:
  - по часовой стрелке;
  - против часовой стрелки.
- Для четных сеток добавлен якорь центра:
  - `NW`;
  - `NE`;
  - `SW`;
  - `SE`.
- Для стандартной сетки `10x10` дефолтный якорь центра остается `NW`: верхний левый из четырех центральных квадратов.
- Режим `спираль от края к центру` заканчивает нумерацию в выбранном центральном якоре.

## Интерфейс

- В список схем `100x100` добавлены спиральные режимы.
- При выборе спирали скрывается настройка `по строкам / по колонкам`.
- Для спирали показывается только релевантное направление: по часовой стрелке или против часовой стрелки.
- Поле стартового угла переключается в режим `Якорь центра`.
- В панели статистики отображаются выбранная спиральная схема, направление и якорь центра.
- Убрана кнопка `Проверить`, потому что проверка и статистика уже обновляются автоматически.

## Проверки

- Добавлены unit-тесты на порядок центральных клеток для спирали `10x10`.
- Проверены оба направления спирали.
- Проверены все четыре центральных якоря `NW/NE/SW/SE`.
- Проверено, что для `10x10` нет дубликатов и пропущенных номеров во всех схемах.
- Добавлен тест KML-экспорта со спиральной нумерацией.

## Что проверить вручную

- Экспорт KML с режимом `спираль от центра наружу`.
- Экспорт KML с режимом `спираль от края к центру`.
- Переключение направления спирали по часовой стрелке и против часовой стрелки.
- Переключение якоря центра `NW/NE/SW/SE`.
- ZIP-экспорт с включенной сеткой `1000x1000` и спиральной нумерацией `100x100`.
- Режим только `100x100` без больших квадратов.
