# v2.1.0

Дата: 2026-04-10
Статус: стабильная версия

## Что изменено

- Зафиксирован итог roadmap `v2.0 -> v2.1`: в приложении появился отдельный пользовательский сценарий `Инструменты` -> `Точки из Excel...` без смешивания point-flow с grid-flow, `.lgm.json`, `GridOptions` и существующей вкладкой `Экспорт`.
- Point-flow поддерживает sample-first `.xlsx` с шаблоном `ФИО | Дата | Координаты`, строгую построчную валидацию, преобразование `X/Y -> zone from Y -> pyproj(y, x) -> lon/lat` и экспорт AlpineQuest-совместимого point-KML с `Placemark` / `Point`, `description`, `ExtendedData/comment` и общим `IconStyle/color`.
- Экспорт point-KML выполняется в отдельном worker-потоке с прогрессом, отменой, проверкой свободного места, записью через временный файл и атомарной заменой результата.
- Окно `Точки из Excel` дополнительно стабилизировано UX-правкой `v2.0.5`: блок `Предварительный просмотр и ошибки` переведен на вертикальный `QSplitter`, а preview/error-панели получили увеличенные минимальные размеры и ручное перераспределение высоты.
- Для релизной подготовки синхронизированы версии в `pyproject.toml`, `src/limuzin_grid_manager/__init__.py` и `version_info.txt`, обновлены `README.md`, `USER_GUIDE.md`, `GRIDBASE.md`, `GITHUB.md` и `versions/GRIDVERSIONS.md`.
