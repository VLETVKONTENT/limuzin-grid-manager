# v2.1.1

Дата: 2026-04-11
Статус: рабочая версия

## Что изменено

- Реализовано безопасное сохранение проектов `Safe Project Save`: `src/limuzin_grid_manager/app/project.py` больше не пишет `.lgm.json` напрямую в целевой файл, а сохраняет JSON во временный файл рядом с проектом, принудительно сбрасывает его на диск и заменяет целевой файл только после успешного `replace()`.
- При любой ошибке до завершения `replace()` временный `.tmp` удаляется, а уже существующий `.lgm.json` остается нетронутым. Это закрывает риск повреждения проекта при неудачной перезаписи.
- В `tests/test_project.py` добавлены регресс-тесты на успешную атомарную запись и на сценарий отказа `replace()`, который подтверждает сохранность предыдущего содержимого проекта и отсутствие мусорных временных файлов.
- Синхронизированы версии в `pyproject.toml`, `src/limuzin_grid_manager/__init__.py` и `version_info.txt`, обновлены `README.md`, `USER_GUIDE.md`, `GRIDBASE.md`, `roadmap.md`, `versions/GRIDVERSIONS.md` и `FIXPROD.md`.
